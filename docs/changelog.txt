[ 0.7 ]

 [NEW] Caching - Caching was added for index.php. This option can reduce server load on highly visited pages.
 [NEW] Debug Mode - A debug mode was added. It will show you some interesting information of what's going on inside the code if something isn't working as it should.
 [NEW] Banlists - Banlists for IP, E-Mail and Domains were added. Choose for yourself who is able to sign your guestbook.
 [NEW] reCaptcha - Full support for Googles reCaptcha.
 [NEW] Dynamic Fieldvariables - Source code names for input boxes in newentry.php are chosen random.
 [NEW] Keystroke - Identify spambots by the speed of typing.
 [NEW] SMTP E-Mails - Send E-Mails from the guestbook over smtp by using phpmailer.
 [NEW] Database backup - Create, restore and delete backups of the guestbook sql tables directly in MGB Admin Panel.
 [NEW] Spam protocol - Every failed attempt to do a guestbook entry (wrong captcha, too fast typing, blocking by banlists, etc.) is logged in Admin panel and can be viewed by Admin.
 [NEW] phpinfo() - See information about your php installation directly in admin panel.
 [NEW] New template "mgbBootstrap" that is using Twitters Bootstrap CSS and is based on html5 by R.W.
 [UPD] The way the date of an entry is displayed was changed to full formatting by date() php function.
 [UPD] Updated the Gravatar support of the guestbook -> "blank" type added
 [UPD] The captcha code is now stored in the session. There was a small chance that a new code was generated by a new user if more than one guests did a new entry at the same time.
 [UPD] Possibility to randomize the length of the security code between 3 and 9 characters.
 [UPD] Completely changed the way the security code is generated. It is generated by using hash algorithms now.
 [UPD] The used hash algorithm can be chosen in admin panel.
 [UPD] A "salt" is mixed into the captcha to make it more random. Can be edited by the user.
 [UPD] Some new options to manage smilies.
 [UPD] No more support for importing entries from an MGB 0.5.x.
 [UPD] If you edit your own user account as an administrator the fields that are not important for you are now deactivated.
 [UPD] Reworked the template of admin panel to make it more well-arranged.
 [UPD] Safer session handling in Admin panel.
 [UPD] Reworked upgrade.php with better upgrade functionality.
 [UPD] New captcha backgrounds.
 [UPD] Updated English, Dutch, French, Schweizerdeutsch and German language files.
 [UPD] MGB Version number is now longer shown on login to admin panel.
 [UPD] Changed the css files of mgbModern. A "main.css" is included in every style now that contains the main settings of the template. All "style.css" do only include colours by now.
 [UPD] New "Facebook" and "Twitter" fields for new entries.
 [FIX] An unique url is added to each captcha which should make sure that it is always newly generated and not loaded out of the cache.
 [FIX] Many bugfixes and optimizations.
